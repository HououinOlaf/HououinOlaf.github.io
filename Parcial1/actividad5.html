# 01_COMPARATIVA_TÉCNICA

Análisis comparativo de los principales marcos de trabajo (Frameworks) y metodologías utilizadas en la industria para la ejecución de pruebas de penetración, auditorías de seguridad y gestión de vulnerabilidades.

# 02_MATRIZ_DE_METODOLOGÍAS

METODOLOGÍA ENFOQUE / OBJETIVO FASES / ESTRUCTURA ORIENTACIÓN VERSIÓN
MITRE ATT&CK Base de conocimiento de tácticas y técnicas adversarias. Detección de ataques. 14 Tácticas (Desde Reconocimiento hasta Impacto). Ataque / Defensa (Blue/Red Team) v16.1
OWASP WSTG Guía de pruebas para seguridad en aplicaciones web y servicios nube. 11 Categorías técnicas de prueba. Evaluación Web v4.2
NIST SP 800-115 Guía técnica de evaluación de seguridad y cumplimiento normativo. Planificación, Descubrimiento, Ejecución, Reporte. Evaluación / Cumplimiento 2008
OSSTMM Manual de medición de seguridad operativa (Métricas tangibles). 5 Canales: Humano, Físico, Inalámbrico, Telecom, Datos. Evaluación Integral v3
PTES Estándar de ejecución de pentesting (Consultoría comercial). 7 Fases: Desde pre-acuerdo hasta el reporte. Ataque / Consultoría v1.1
ISSAF Framework de evaluación profunda de sistemas (Auditoría compleja). Planificación, Evaluación, Reporte, Limpieza. Evaluación de Sistemas v0.2.1

// NOTAS_ADICIONALES

  • > MITRE: Utilizado ampliamente por SOCs y Threat Hunters.
  • > OWASP: El estándar de facto para seguridad web (Certificaciones OSWE/GWAPT).
  • > PTES: Enfocado en estandarizar la calidad del servicio de pentesting.
