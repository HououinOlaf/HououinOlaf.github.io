# 01_MARCO_INTRODUCTORIO

[cite_start]

El objetivo central de este laboratorio consistió en el despliegue de una Red Privada Virtual (VPN) utilizando el conjunto de protocolos IPSec[cite: 140].

[cite_start]

El escenario se recreó mediante Cisco Packet Tracer para establecer un canal de comunicación cifrado entre dos sucursales remotas, permitiendo que la información viaje segura a través de una red pública (ISP)[cite: 141, 142].

isp(config-if)# ip add 209.165.200.2 255.255.255.0
isp(config-if)# no shutdown
isp(config-if)# exit
% LINK-5-CHANGED: Interface GigabitEthernet0/0, changed state to up

# Definiendo tráfico VPN (ACL 110) R1(config)# access-list 110 permit ip 192.168.1.0 0.0.0.255 192.168.3.0 0.0.0.255

# Creando el Mapa Criptográfico R1(config)# crypto map MAPA_VPN 10 ipsec-isakmp
